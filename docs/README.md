# Polybot Documentation 📚

This directory contains the primary design and requirements documents for Polybot v1.0.

- **`Polybot_PRD (1).pdf`**: The Product Requirements Document. Details the core agent functions, data flows, user stories, and features required for the autonomous trading engine.
- **`Polybot_System_Design.pdf`**: The technical architecture and system design specification. Includes diagrams and detailed API sequences for interaction with the Polymarket and Gamma interfaces.

For setup and installation instructions, please refer to the main [`README.md`](../README.md) in the project root.
